# gcb

Copy image or text into your clipboard, from the terminal!

# Downloads

Binaries available in github [releases][https://github.com/thewh1teagle/gcb/releases/tag/latest]

# Usage

### Image
```console
gcb image.jpg
```

### Text
```console
gcb "Paste this text into clipboard!"
```

### Stdin
```console
echo "hello" | gcb 
```
